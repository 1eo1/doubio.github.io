鉴于最近实在没什么教程好写的，再加上有不少人想要让我写一个建立博客的教程，那我干脆就录制一个 VPS上安装Wordpress的视频教程好了。

因为我做视频经验不多，想到哪做到哪 一遍过，视频中出现了一些错误，所以后期稍微剪辑了一下，但是剪辑技术很菜所以可能剪辑的地方会不太流畅（鬼畜），不过影响不大，大家包涵。

点击下面的链接，可以直接在线观看，如果逗比云网络炸了或者想要本地观看的，请右键链接另存为本地文件。

视频链接：

逗比 · 云

时长：15分钟。

视频教程使用的是 Debian 7 X64 系统，请尽量使用这个系统，否则可能部分步骤有差别导致安装失败！

教程使用的是 Vultr 的VPS，使用的是LNMP一键包，同时关于LNMP一键包Wordpress不显示主题 和 上传图片需要填写FTP 等等问题一同演示了。

注意：视频教程中的所有设置密码的我都设置为 Toyo233 ，分别是：数据库账号的ROOT账号的密码（注意是数据库的ROOT账号 而不是VPS的ROOT账号）、数据库账号的WP账号的密码、Wordpress博客管理员账号的密码

视频中的解决wordpress主题不显示的代码：

sed -i 's/,scandir//g' /usr/local/php/etc/php.ini
# 取消PHP scandir函数禁用
/etc/init.d/php-fpm restart
# 重启PHP
chmod 777 -R /home/wwwroot/xxx.xx/wp-content
# 赋予文件夹权限，注意替换 xxx.xx 为自己的域名

视频中的解决wordpress上传主题/插件提示输入FTP账号的代码：

添加到 wp-config.php 文件最后。

define("FS_METHOD", "direct");
define("FS_CHMOD_DIR", 0777);
define("FS_CHMOD_FILE", 0777);
define('CONCATENATE_SCRIPTS', false);

phpmyadmin ROOT密码输入正确，但是提示：无法登陆 MySQL 服务器

这是lnmp 1.3 一键包的一个BUG，只要重置一下数据库ROOT密码就好了。

假设你当初安装lnmp时，是在root文件夹执行的，那么进入目录(如果不一样，自己替换目录)

cd /root/lnmp1.3-full/tools

然后执行 数据库ROOT密码重置脚本，会提示你输入新的数据库ROOT密码。

./reset_mysql_root_password.sh
