AMH5.3开心版本

系统支持

CentOS / Ubuntu / Debian

特别说明

AMH5.3安装完毕之后,会出现登陆一片空白请登陆面板点开左边工具栏的“数据同步 更新缓存数据” 选择更新缓存数据 之后才会显示数据。

开心说明

  1. 去除授权，去除文件完整性验证。
  2. 可无限次下载所有模块。
  3. 由于模块下载是通过我的服务器，可能不是最新的，但是我会时常更新保持和官方一致（如果觉得不可信请不要用，没人逼迫！）
  4. 最近我发现有许多人造谣我的脚本有后门,我就笑笑.还是那句话 爱用不用！
  5. 不推荐CentOS 7系统安装AMH，详细请去官网看系统支持说明。

优化说明

安装时可自定义AMH and Mysql密码，不再是随机生成。

安装&卸载

wget http://amh.janch.top/YD-panel.sh && bash YD-panel.sh 2>&1 | tee YD-panel.log

wget http://amh.ydidc.top/YD-panel.sh && bash YD-panel.sh gcc 2>&1 | tee YD-panel.log

#备用
wget http://amh.janch.top/YD-panel.sh && bash YD-panel.sh gcc 2>&1 | tee YD-panel.log

懒人版：

懒人版说明：全自动安装 php7.0\lnmp\pure-ftpd\amftp\phpmyadmin 等建站必备的模块，无需再一个一个下载安装。

懒人版 极速安装：

AMH5.3 卸载

所有数据都会删除。注意有数据请先备份

killall php-fpm
amh nginx stop
amh mysql stop
rm /root/amh -rf;
rm /home/usrdata /home/wwwroot -rf;
rm /usr/local/amh* -rf;
rm /usr/local/libiconv* -rf;
rm /usr/local/nginx* -rf;
rm /usr/local/mysql* -rf;
rm /usr/local/php* -rf;
rm /etc/init.d/amh-start /etc/amh-iptables /bin/amh -f;
rm -f amh.sh
