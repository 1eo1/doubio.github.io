一、firefly扫盲

firefly，全称为Firefly-proxy，也就是萤火虫代理，是一个有轮子背景的翻墙软件，类似于free门，无JIE，不过free门，无jie自2014年11月开始一直封锁的厉害，近期几乎处于瘫痪状态，无法使用，然而firefly却是出奇的稳定，老版的0.2.0至今依旧可以使用，这是因为它内置了Tor 浏览器才有的的meek插件，Meek 插件使用了一种叫做“domain fronting”（域名欺骗）的流量混淆技术，这种方法的特殊之处在于，翻墙时的请求指令中表面上请求一个域名，而实际上将真实请求的域名藏在 HTTP 的 Host header 之中，只有代理服务器才可以识别隐藏的域名指向，而Tor 将代理服务器搭建在几个世界上最大的主机服务商例如 Google App Engine、Amazon CloudFront 或 Microsoft Azure，GFW要屏蔽上述网站，所要付出的代价是相当沉重的。

当然了，赵子龙今天写这篇文章不是要小伙伴们用老的0.2.0版本，因为firefly0.3.0发布了，由于早期的版本有速度慢，特定网站（如DW,GOOGLE）打不开的问题，再加上轮子很少宣传firefly，所以知道有这个软件的人不多，0.3.0修复了以上两个问题和一些bug，翻墙速度大大加快，目前仅支持 Windows 系统

二、firefly使用

firefly没有软件官网，想要下载必须到他的github页面去，所幸github目前没有被墙，这里给出地址

https://github.com/bannedbook/fanqiang/wiki/%E8%90%A4%E7%81%AB%E8%99%AB%E4%BB%A3%E7%90%86

下载链接在最下方，如图所示，推荐点击一键下载，因为下载2需要BitTorrent Sync支持

下载完成后会得到如下图所示的压缩包，名称为firefly-proxy-win，大小为5.35MB，由于是压缩包，所以数字签名就不用检查了

双击并解压到文件夹，可见很多文件，双击图中圈出的firefly.exe即可运行代理

运行后它会在任务栏显示一个萤火虫图标，右键点击可以选择打开内置的chrome浏览器，配置代理，退出

 

之前已经说过，firefly有轮子背景，所以它自带的浏览器一打开就会显示你懂的内容，这个内容弹出下文会给出关闭的方法，当然了，这个软件是开源软件，在项目主页上代码都是公开的，所以你不用担心这个程序有什么恶意功能。

下面简单说一下配置代理，配置代理是firefly的高级选项，由于firefly打开即可使用，小白可以跳过此部分不看，

打开配置代理页面后如图所示，白名单黑名单，内置host的选项写的很清楚，如果想要更改点击即可，通常不建议去动

点击左上角的代理配置后会看到如下图第一个红圈所示的“基于shadowsocks的sock5代理”选项，选择此项并填写你的SS账号可以直接让内置的浏览器调用你的SS代理，而不需要开shadowsocks，非常方便，TCP FAST OPEN不建议勾选

如果要修改主页看到下图第二个圈，输入你想设置的主页后点击保存后即可生效，设置后就不会再打开轮子的主页，这里个人比较推荐这个网址导航 ——http://hao.4a8a.com/

如果选择左边的hosts配置（如下图所示）你可以点击查看全部选项，然后复制hosts列表到自己的系统或分享给别人

更新内置host文件选项就不用点了，这个通常都是最新的

三、注意

  • 翻墙之前务必关闭360安全卫士，腾讯管家等国产安全软件及其所有后台进程，否则容易被切断代理线路或被监控
  • 不要点击左菜单区的分享给朋友选项，如果要这么做，请在不要在分享页面填写国内邮箱，诸如QQ邮箱，,SINA邮箱
  • 有时页面延迟较高打不开，请刷新或重启firefly
