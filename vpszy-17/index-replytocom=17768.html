单向流量说明

因为Shadowsocks/R的原理实际上是一个流量转发工具，所以如果使用Shadowsocks/R下载了大小为 100MB的文件，实际上是VPS下载了文件(100M)，然后上传给你(100M)，共走了200MB流量，而如果流量是单向计算的话，IDC只会计算上传流量，所以只会显示消耗了100MB的流量。

买VPS前记得分清楚是不是单向流量计算的，如果是双向的，做Shadowsocks/R代理的话就需要减半去看。

注意事项

注意：这家VPS购买后，默认没有安装系统，需要自己重装系统才可以用，否则会Ping不通和SSH无法连接！

另外，此IDC的香港VPS，虽然VPS内分配了IPv6地址，但是实际上没有IPv6网络，所以在有时候(比如安装软件 apt-get / yum)会先尝试IPv6然后卡住，禁止IPv6即可。

# 临时禁止：
echo 1 > /proc/sys/net/ipv6/conf/eth0/disable_ipv6

# 临时启用：
echo 0 > /proc/sys/net/ipv6/conf/eth0/disable_ipv6

# 永久禁止：
echo "net.ipv6.conf.all.disable_ipv6 = 1" >> /etc/sysctl.conf
sysctl -p

# 永久启用：
sed -i '/net.ipv6.conf.all.disable_ipv6/d' /etc/sysctl.conf
sysctl -p

  1. 首先进入 后台管理面板，然后如下图所示，选择要升级套餐的服务。
  2. 进入服务管理页面后，在左侧的操作栏中，点击 升级 / 降级 选项。
  3. 进入服务升级/ 降级 页面后，可以看到新的 A1、A2、半年优惠套餐，根据你当前的服务套餐来找到对应的新套餐，并点击 选择产品。

  5. 最后就会提示你订购成功，这时候回到一开始的 服务管理页面，并重装VPS（务必），即可应用最新的配置了！
